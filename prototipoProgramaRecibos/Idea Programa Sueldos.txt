Programa liquidacion de sueldos(Liquid's):
	
	Funcion de programa: 

	excelEmpleados --> programa -->  excelAImprimir
	
	El programa tomaria ya los datos de los empleados de varios excel (un excel por empleado) los cuales ya tendrian los datos de los empleados necesarios y dejaria otros excel's armados para directamente imprimir los recibos de cada empleado de cada mes en una carpeta.

	Comentarios: Si optamos por un excel por empleado podria hacer un programa para que desde excelEmpleados (existente) tome todos los datos de cada uno y los separe en diferentes .txt o .xlsx (o simplemente pasarlos a mano) asi dependiendo el legajo se tomen los datos mas rapido de cada empleado, este todo mas ordenado, y para que el programa sea mas eficaz y rapido.
	Cada excel asignado a cada empleado podria contener el legajo y el cuil, asi se podria generar la busqueda mensionada mas abajo.

	Diseño: Podria ser igual a la hoja del recibo, asi es mas facil que el usuario se guie y comprenda el programa.

	Variables/Constantes {

		-companyData: Un simple selector que de como opciones las diferentes empresas

		-employeeData:Ingresando el Legajo o el Cuil, se tomaran automaticamente los valores de todas las variables {
			-Legajo
			-Cuil
			-Sector
			-Tarea
			-Fecha ingreso
			-NombreApellido
			-Fecha deposito
			-Periodo
			-Categoria
			-nombreBanco			
			-RemuBasica
			-TareaAsignada
		}

		/Ingresado por el usuario/{
			-NroRecibo
			-LugarFecha de pago
			-Unidades
			-Valor unitario
			-Descripcion			
			-HaberesRemunerativos/ noRemunerativos
			-Descuentos
			-Aumentos: Si hay entonces:
				-Remunerativo: el usuario ingresa el porcentaje y se actualiza el monto a sumar
				-No remunerativo: Mismo que el otro
			-Antiguedad: Usuario ingresa antiguedad
		}
		

		-Periodo abonado: Podria ser un simple selector el cual le de a elegir el mes

		-Subtotal: Calcula el subtotal

		-TotalNeto: Calcula el total neto

		-TotalAEscrito: Convierte los numeros de totalNeto a letras
	}
